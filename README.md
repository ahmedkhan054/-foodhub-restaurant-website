Online Restaurant Website (PHP + MySQL)

A fully responsive, modern online restaurant website that allows customers to browse the menu, search and filter food items, add to cart, place orders, and request home delivery. It includes promotional popups, flash sales, customer testimonials, and an engaging UI.

Features

User Features
- Browse food menu with category filters (Pizza, Burger, Pasta, etc.)
- Search food items via search bar
- View detailed food information (Food Details page)
- Add/Remove items from shopping cart
- Place orders with home delivery option
- Contact form with Google Maps integration
- Customer reviews and testimonials slider
- Social media links and opening hours

Marketing & Engagement
- Welcome popup on site load
- Promotional offer popup (25% OFF on first order)
- Flash sale countdown timer
- Limited-time offer banner
- Buy One Get One Free promotion section
- Exit-intent discount popup
- Newsletter subscription

UI/UX
- Sticky header with smooth scroll
- Hover animations on buttons and cards
- Fade-in and slide-in animations (AOS library)
- Animated counters
- Image hover effects (zoom and overlay)
- Loading spinner
- Fully responsive (mobile‑first design)

Technology Stack

- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript (jQuery)
- **Libraries:** AOS (scroll animations), Swiper.js (testimonials slider), Font Awesome 6
- **Backend:** PHP 7.4+
- **Database:** MySQL 5.7+
- **Server:** Apache / Nginx (with `.htaccess` support)
